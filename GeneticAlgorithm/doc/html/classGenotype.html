Detailed Description

template<typename T, template< typename, typename=std::allocator< T >> class C = std::vector>
class Genotype< T, C >

Genotype template used to store genes values.

Genotype template can be instantiated interchangeably using only the type of stored genes (ex. Genotype<int>) that defaults to Genotype<int, std::vector>, or it can be instantiated with collection type, for example Genotype<int, std::list>

Ccollection that stores genes
